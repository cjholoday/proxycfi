**************************************************************************
To use the alternate dynamic linker do either of this two:
1. Excplicitly invoke the dynamic linker to run the program:
./path/to/alternate/ld-linux.so.2 program [args]
2. Use this flag during linking:
-Wl,-dynamic-linker,/path/to/alternate/ld-linux.so.2
